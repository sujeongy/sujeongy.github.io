# Terminal Basic

자주 쓰는 터미널 명령어 모음.

## 파일 및 폴더 관리

### 폴더 생성
```bash
# newFolder라는 이름의 폴더 만들기
mkdir newFolder
```

### 파일 생성
```bash
# 빈 텍스트 파일 생성
touch hi.txt
```

### 파일 삭제 (주의 ⚠️)
```bash
# 강제로 파일/폴더 삭제 (복구 불가)
rm -rf folderName
```

---
## 실행 및 열기 (Workflow)

터미널에서 현재 작업 중인 폴더를 다른 프로그램으로 바로 열 때 사용합니다.
여기서 `.` 은 **현재 폴더**를 의미합니다.

### Finder 열기 (macOS)
현재 위치를 파인더(탐색기) 창으로 엽니다.
```bash
open .
```

### VS Code 실행
현재 프로젝트를 VS Code로 바로 엽니다.
```bash
code .
```
> 만약 터미널에서 command not found: code라고 뜬다면, VS Code 설정이 안 된 것  
> VS Code에서 Cmd + Shift + P 누르고 **"Shell Command: Install 'code' command in PATH"**를 실행해야 작동

### 파일 실행
특정 파일을 기본 연결 프로그램으로 실행합니다.
```bash
# 예: index.html을 브라우저로 열기
open index.html
```
