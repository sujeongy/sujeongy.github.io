# VS Code Shortcuts (Mac)

마우스를 쓰지 않고 코딩 속도를 높여주는 필수 단축키 모음.

## 🚀 네비게이션 (Navigation)
* **`Cmd + P`**: 파일 이름으로 빠르게 찾기 (Quick Open)
* **`Cmd + Shift + P`**: 모든 명령어 검색 (Command Palette)
* **`Cmd + Shift + F`**: 전체 파일 내용 검색

## ✍️ 편집 (Editing)
* **`Opt + ↑ / ↓`**: 현재 줄 위/아래로 이동 (가장 많이 씀 👍)
* **`Opt + Shift + ↑ / ↓`**: 현재 줄 위/아래로 복사
* **`Cmd + Enter`**: 커서가 문장 중간에 있어도 바로 아래에 새 줄 추가
* **`Cmd + /`**: 주석 처리 토글

## ⚡️ 멀티 커서 (Multi-Cursor)
* **`Cmd + D`**: 드래그한 단어와 같은 단어를 하나씩 추가 선택
* **`Cmd + Shift + L`**: 같은 단어를 문서 전체에서 한 번에 선택
* **`Opt + Click`**: 클릭하는 위치마다 커서 추가

## 🖥️ 화면 제어 (View)
* **`Cmd + B`**: 사이드바 열기/닫기 (코드 창 넓게 보기)
* **`Cmd + \`**: 에디터 화면 분할 (Split)
* **`Ctrl + ~`**: 하단 터미널 열기/닫기
